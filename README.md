# 2019-Jul-HotDog

# API for the Hot Dog app

## Initiate API

Install required packages: 

    npm install

Begin Server:

    node index.js

## Get All Dogs

Make ```GET``` request to ```/dogs```

Responds with json:

    {
        success: bool,
        message: 'XXXX',
        data: object (all Dogs info)
    }

## Update Winner

Make ```POST``` request to ```/dogs/:id/win```

where :id is the MongoDB _id
